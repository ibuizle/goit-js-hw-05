# JavaScript OOP: Classes & Prototypes 🏗️

This repository contains the fifth homework assignment for the GoIT Fullstack Developer course. It focuses on the principles of **Object-Oriented Programming (OOP)** in JavaScript, including constructors, prototypes, and the `class` syntax.

🔗 **[Live Demo](https://ibuizle.github.io/goit-js-hw-05/)**

## 📖 About the Project

The purpose of this assignment was to master the creation of objects and the management of their logic through methods. The tasks cover the transition from functional programming to OOP, emphasizing code reusability and clean structure.

### Key Learnings & Features:
- **`this` Keyword:** Understanding the execution context and how `this` refers to different objects depending on how a function is called.
- **Prototypes:** Learning how JavaScript objects inherit properties and methods through the prototype chain.
- **ES6 Classes:** Implementing modern `class` syntax to create blueprints for objects, including constructors and private/public methods.
- **Method Chaining:** Building classes where methods can be called one after another for cleaner logic.
- **Inheritance:** Practicing how to extend classes and share functionality between different object types.



## 🛠 Built With

- **HTML5:** Basic structure for testing scripts.
- **Vanilla JavaScript:** ES6+ features, specifically focusing on Classes and Objects.
